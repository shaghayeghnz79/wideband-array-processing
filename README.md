# Wideband Array Processing with Interference Mitigation

This project presents the implementation and evaluation of a wideband microphone array processing pipeline designed for a moving acoustic source in the presence of noise and interference. The goal of the project is to model realistic sound propagation, estimate time-varying delays between microphones, align the received signals, and improve signal quality using spatial processing techniques. The system simulates a moving sound source following an elliptical trajectory while modeling physical propagation effects such as time-varying delays and distance-based attenuation.

The first stage of the project focuses on modeling the acoustic propagation from the moving source to the microphones. The received signals are generated by applying time-varying propagation delays and inverse-square attenuation to the original source signal. These simulated microphone signals serve as the basis for analyzing delay estimation accuracy and the effect of source motion on signal reception.

The second stage addresses the estimation of the differential time of arrival (DToA) between microphones using short-time cross-correlation. Since delay estimates may fluctuate due to noise, temporal smoothing is applied to stabilize the delay trajectories over time. The estimated delays are then used to align the signals from multiple microphones so that the desired source becomes temporally synchronized across the array.

Once alignment is achieved, beamforming techniques are used to combine the microphone signals and enhance the desired source. Both simple delay-and-sum beamforming and statistically optimal combining methods such as BLUE/MVDR are evaluated. These approaches exploit spatial diversity in the array to improve the signal-to-noise ratio by reinforcing the coherent desired signal while reducing incoherent noise.

The final stage of the project investigates interference cancellation in scenarios where some microphones are contaminated by an interfering signal. Several strategies are examined, including non-adaptive reference subtraction, adaptive noise cancellation using the NLMS algorithm, and covariance-based BLUE/MVDR combining after alignment. The performance of these methods is evaluated using metrics such as signal-to-interference ratio, correlation with the desired signal, and mean squared error.

Experimental results demonstrate that accurate delay estimation and signal alignment are crucial for effective array processing. Beamforming significantly improves the output signal quality compared to a single microphone, and adaptive or covariance-based approaches provide further improvements when interference is present. Overall, the project illustrates how delay estimation, beamforming, and adaptive filtering can be integrated to enhance wideband audio signals in realistic acoustic environments.
